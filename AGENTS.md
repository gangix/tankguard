# TankGuard repository instructions

- Read `PROJECT_SPEC.md` before making changes.
- Stack: one Next.js App Router application written in TypeScript with Tailwind CSS; SQLite is the local database.
- Keep out of scope: authentication/multi-tenancy; CRUD for seeded records; real GPS or fuel integrations; file uploads; and mobile work beyond basic responsiveness.
